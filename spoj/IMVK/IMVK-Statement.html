Its the final match between India and Pakistan. Pakistan scores a huge total (say "N") in his innings. Since its a big score, all eyes are set on starplayer "Virat Kholi". Virat loves to chase big scores. But Anushka says "he can only chase this score if total is divisible by 495."
MS Dhoni is weak in maths. Now your task is to inform MS Dhoni whether N is a multiple of 495 or not.

Constraints:


0 < T < 11
0 <= N < 10^10^5 (same as 10^100000).

Input

First line of input will contain an integer T which denotes the number of testcases. Each of next T lines cointains an integer number N.

Output

For each of the number N, you have to print yes, if that number is a multiple of 495,otherwise print no.

Example

Input:
2
2
990
Output:
no
yes
